### 单例模式
### Singleton Pattern

<br>

##### 确保一个类只有一个实例，并提供一个全局访问点。

<br>

##### Ensure a class has only one instance, and provide a global point of access of it.

<br>

> 很多对象只能存在一个，如果制造出多个实例，反而会导致许多问题的产生，例如线程池，缓存，对话框，注册表对象等。

<br>

> There are many objects we only need one, if we were to instantiate more than one we'd run into all sorts of problems.Like thread pool,cache ,dialog boxes, registry settings...etc
